The page isn't displaying, what's wrong?

Since D&D Buddy is hosted on free servers, they "fall asleep" after a short period of time of no use. When someone wakes up the servers the first request takes much longer than normal requests. Simply wait a few moments while the servers wake up and try again.

I joined a game as a DM, but can only see the roll panel, why?

If you join a game as a DM and there is no one else in the game with you, then you cannot to make changes to characters until the join the game. If you need to make changes to a character, go to the DM Portal.

I don't see the game I want to join when creating a character

If you are unable to join the game you want, make sure that you don't already have a character in the game and that the DM doesn't have the game locked.
